# 🚀 OmniScroll Live

**OmniScroll Live** is a premium, lightweight WordPress ticker plugin designed for high-performance live streams and modern websites. It features a seamless smooth-scroll engine, real-time admin previews, and deep customization via Hex color pickers and typography controls.

## ✨ Key Features

* **⚡ Real-time Preview:** See your changes instantly in the admin dashboard before saving.
* **🎨 Hex Color Precision:** Full control over background, dots, and button colors using integrated color pickers.
* **📏 Dynamic Sizing:** Adjust strip height and font sizes for a perfect fit on any layout.
* **🖱️ Hover Interaction:** Optional "Pause on Hover" functionality for better user engagement.
* **🔗 Action Buttons:** Add call-to-action buttons with custom icons (FontAwesome) inside the scroll.
* **🔄 Auto-Updates:** Integrated GitHub update checker—get the latest features directly from GitHub.

## 🛠 Installation

1.  **Download:** Download the latest `Source code (zip)` from the [Releases](https://github.com/simba963/omniscroll-live/releases) page.
2.  **Upload:** Go to your WordPress Dashboard > **Plugins** > **Add New** > **Upload Plugin**.
3.  **Activate:** Click **Activate** after the upload is complete.
4.  **Configure:** Navigate to the **OmniScroll Live** menu in your sidebar to build your first ticker.

---

## 📖 Usage

After configuring your settings in the admin panel, simply use the following shortcode anywhere on your site (Pages, Posts, or Widgets):

`[omniscroll]`
